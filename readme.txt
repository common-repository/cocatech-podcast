== Description ==
Esse plugin cria um Widget que exibe a lista dos 5 últimos episódios do Cocatech Podcast no local que você escolher em sua sidebar, footer ou header.
Cocatech é um podcast diário sobre tecnologia, desde um ponto de vista Apple.

http://www.cocatech.com.br

Caso você curta o Cocatech e tenha um site em WordPress, instala o plugin vai! =)
== Installation ==
Para instalar é simples:

Instale diretamente do seu WordPress indo em plugins e procurando por Cocatech.
Ou faca o download, suba para wp-content/plugins usando um cliente ftp e ative no seu painel WordPress.
